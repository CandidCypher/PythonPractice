# Data Strucutres

<p> There are two categories of data strcutres; <i>primitives</i> and non <i>primitive</i> types. 

* Primitive types
** float
** int
** char
** string
** double
** long


* Non Primitive types (composed of primitives)
** Arrays
** Lists / Arrays
** Sets
** Maps
** Dictionaries
** Graphs
** Linked Lists
** Trees
